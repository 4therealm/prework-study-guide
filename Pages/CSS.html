Rules

  • a class or id name cannot start with a number
  • (*) is a universal selector

Syntax

    selectors

  • simple = select elements based on name, #id, or .class
  • combinator = select elements based on specific relationship between them
  • pseudo-class = select elements based on a certain state
  • pseudo-element = select and style a part of an element
  • attribute = select elements based on an attribute or attribute value

    Margin & Padding

  • margin: 0px 0px 0px 0px (top, right, bottom, left)
  • margin: auto; = horizontally center the element within its container
  • margin: values = auto(browser calculates), length(specifies margin in px, pt, cm, etc), %(specifies a margin in % of the width of the containing element), inherit(margin should be inherited from the parent element)

    Links

  • a:link = normal, unvisited link
  • a:hover = a link when the user mouses over it
  • a: = a link the user has visited
  • a:a link the moment it is clicked
  • a hover must come after link and visited
  • active must come after a hover
