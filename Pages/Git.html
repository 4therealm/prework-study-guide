Git commands

  • git status = shows modified files in your working directory, staged for next commiy
  • git add -A = add all changes to staging area
  • git commit -m "descrptive message of changes"
  • git branch [branch name] = creates new branch
  • git checkout [branch name] = switch to other branch and put in working directory
  • git checkout -b [new branch name] = creates new branch and moves it into working directory
  • git innit = initialize and existing directory as a git repository
  • git clone [url] = retireieve entire repository from a hosted location via url
  • git pull = pull any changes from the remote branch
  • git push origin main = transmit local branch commits to the remote repository branch
